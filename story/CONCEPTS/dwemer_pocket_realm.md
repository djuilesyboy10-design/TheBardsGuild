CORE IDEA

Alduin’s name cannot be spoken directly.
It must be reconstructed.

Not because it’s forbidden —
but because names like his only work when remembered from multiple angles.

A single culture remembers fear.
Another remembers record.
Another remembers function.

Only when those fragments align does the name regain causality.

That’s beautiful, and very Elder Scrolls.

THE RIDDLE STRUCTURE (THE HIDDEN NAME)

The riddle does not say “Alduin.”
It describes what happens when his name is present.

Something like (conceptually, not final text):

When the song has no last note,
When fire waits for permission,
When gods survive what should have ended them —
What name was forgotten?

Each NPC answers part, not the whole.

THE THREE (FOUR) KEEPERS OF THE NAME
1. Hisin the Nord — Myth / Fear / Oral Memory

Hisin carries:

the emotional truth

Alduin as inevitability

Alduin as devourer

He remembers:

“The name that makes even gods prepare to die.”

He cannot define it.
He only knows what it does to people.

This anchors Alduin in Nordic myth, without repeating Skyrim prophecy.

2. The Orc Record Keeper (Valenvaryon) — Function / Record

This is crucial and very clever on your part.

The Orc doesn’t care about dragons.
He cares about:

outcomes

cycles

annotations that never resolved

He has a record that:

lists a world-ending process

shows it was interrupted

has no closing mark

To him, Alduin is not a being —
he is an unfinished procedure.

This ties directly to:

Mora

archives

“If it’s written, it must have happened”

Except here, it didn’t finish.

3. Yfga the Nord (Pelagiad) — Memory / Witness

Yfga is perfect as the final revealer, not because she’s wise, but because she’s human.

She doesn’t know lore.
She remembers a rhyme, a children’s cadence, a half-forgotten song.

Something Alduin-adjacent that survived because:

it wasn’t important enough to erase

it wasn’t correct enough to be dangerous

This is where:

mindok los silaan
fails.

Because memory persisted sideways.

Yfga doesn’t “solve” the riddle.
She completes its rhythm.

Optional 4. The Player (Lore Keeper) — Synthesis

The player does the only thing no NPC can:

They hold:

fear (Nord)

function (Orc)

memory (Yfga)

And realize:

the answer is not a word,
it’s a name that demands an answer.

That’s when ALDUIN can be invoked — not spoken aloud yet, but acknowledged.

HOW THIS TIES BACK TO THE DRAGON MEMORY QUEST

When the player enters Alduin’s memory pocket:

Alduin already knows his name

But the world has forgotten how to answer it

The riddle quest ensures:

the player understands what answering means

the player has already allowed an ending once

So when Alduin looks at the player and listens, it’s earned.

WHY THIS IS STRONG DESIGN

No exposition dumps

No prophecy rehash

Cultures disagree without contradiction

Mora’s archive remains “correct”

The Hist still whisper through persistence

Most importantly:

Alduin is restored by remembrance, not release.
